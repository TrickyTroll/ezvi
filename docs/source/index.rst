Welcome to ezvi's documentation!
=================================

.. image:: ./img/ez_vi.svg
  :alt: The ezvi snek

``ezvi`` is a python package that allows typing automation for the ``Vi`` editor.
Simply provide a YAML file telling it what to do, and ``ezvi`` will type the file
for you. **It even knows how to exit!**
